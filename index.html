Modules

CSS - Images - JS 


var CSS - require(‘css!./file.css’);

Modules

Load a file on-demand or load with a promise 


var onDemand - require.ensure([“mod-a”], function(require) {
  var mod-a = require(‘mod-a’);
});
var Promise - require(‘promise?es6-promise!./file.js’);

Configuration

  • Concat/Minify many files into a single AMD/UMD/CommonJS file
  • Entry Points and code-splitting/chunks
  • Loading Chunks on-demand (require.ensure)
  • Use with grunt/npm/gulp/karma
    • Pass webpack.config.js

Configuration

webpack.config.js 


...
output: {
  path: 'public/js',
  filename: '[name].js',
  publicPath: 'js',
  libraryTarget: 'amd'
}
...
entry: {
  App: ['./app/index.js']
}

Configuration

webpack.config.js 


...                
loaders: [
  { 
    test: /\.js$/, 
    exclude: /node_modules/, 
    loader: 'babel-loader?experimental&optional=runtime'
  }
]
...
